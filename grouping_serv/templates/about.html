The algorithm

The website collects data of favored and unfavored groups from each classroom user

and form a datastructure: [user, [[favored, []], unfavored, []]]

the program calculate the all possibility of grouping of every member in a class

Happiness calculating algorithm

In each group, if one person is grouped with his/her favored person, overall happiness of the group +1,

if grouped with unfavored person, overall happiness of the group -1.

The difference of the most happy group and the most unhappy group is measured in every cases,

The algorithm finally randomly select the case with the happiest least happy group.
